### 🔄 **Restart ระบบ:**

```bash

```

### 📊 **ตรวจสอบสถานะ:**

```bash

```

### 📝 **ดู Logs:**

```bash
ssh root@203.170.129.199 "cd /opt/internship-system && docker logs internship_backend --tail 20"
```
